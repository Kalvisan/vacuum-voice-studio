# Voice file map — Xiaomi X20 Max (d109gl)

The official English voice pack contains **101 MP3 files** named by number only (for example `720.mp3`).
Each number is a fixed slot on the robot. Replace a file to change what the robot says in that situation.

**Robot model:** `xiaomi.vacuum.d109gl` (Xiaomi Robot Vacuum X20 Max, also sold as X20 Max)

## Quick guide

1. Find the phrase or situation below.
2. Note the file name (for example `720.mp3`).
3. Save your replacement MP3 with the **exact same name**.
4. Replace the matching file in Voice Pack Studio (drag-and-drop).
5. Build and install the pack with this tool.

CSV for scripts and AI agents: [assets/d109gl_en_transcriptions.csv](../assets/d109gl_en_transcriptions.csv)

## Rules

- Keep all 101 file names. Do not add or remove files.
- Only replace the files you want to change; leave the rest as downloaded.
- File `01.mp3` and `02.mp3` are silent/blank slots (no spoken phrase). Whisper misheard them as ads; leave them empty or unchanged unless you know you need them.
