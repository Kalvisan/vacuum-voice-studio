# Contributing

Contributions are welcome. Please help keep the project **transparent, privacy-respecting, and easy to trust**.

Please follow these rules:

- Do not commit Xiaomi audio files.
- Do not commit third-party audio unless you have the rights to share it.
- Do not commit account tokens, device tokens, signed URLs, captures, or private logs.
- Keep user-facing text in English.
- Prefer clear error messages that explain what the user can do next.
- Test ZIP validation before changing install logic.
- Run `python3 scripts/repo_virusscan.py` before opening a pull request.

Useful contribution areas:

- support for more Xiaomi/Dreame model file tables,
- clearer session setup instructions,
- clearer TUI flows,
- better validation of MP3 format and duration,
- translations of documentation in separate files.
